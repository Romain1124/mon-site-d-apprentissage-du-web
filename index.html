Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Le web, c'est un immense espace connecté où l'on peut explorer, apprendre et partager en quelques clics.
IL nous permet d’accéder à une quantité infinie d’informations, et repose sur des technologies comme le HTML, le CSS et le JavaScript, et il est consultable à l’aide de navigateurs web.
C'est un ensemble de pages, accessibles depuis un lien hypertexte comme une toile d'araignée " World Wide Web" , soit « toile mondiale, créé en 1989 par Tim Berners-Lee.

Internet

Internet, c’est le réseau mondial qui connecte des milliards d’appareils et permet la circulation des données entre eux.
Il est la base de nombreux services comme le web, les e-mails ou encore le streaming, et fonctionne grâce à un ensemble de protocoles, dont le plus connu est le TCP/IP.
Sans Internet, notre monde serait bien différent, car il est devenu essentiel pour communiquer, travailler et s’informer au quotidien.

Cloud

Le cloud, c’est comme un disque dur géant dans le ciel, où je peux stocker mes fichiers et y accéder de n’importe où.
C’est super pratique pour ne rien perdre et retrouver ses documents, photos ou projets sur plusieurs appareils.
En plus, ça permet d’utiliser des outils puissants sans avoir un ordi ultra-performant juste une connexion Internet, ce qui change vraiment la façon de travailler et de partager des infos.

W3C

Le W3C, c’est un peu le gardien des règles du web, celui qui s’assure que tout fonctionne bien et de manière harmonieuse.
C’est un groupe d’experts qui définit les standards pour que les sites et les technologies web soient accessibles, compatibles et évolutifs.
Grâce à eux, le web reste un espace ouvert et utilisable par tous, peu importe l’appareil ou le navigateur utilisé.

Partie 2 - Distinguez les langages du Web

Langages client

Les langages client sont des langages informatiques qui s’exécutent directement sur l’ordinateur ou le téléphone du client, sans passer par un serveur.
Ils servent principalement à afficher et animer les pages web, comme le HTML pour le contenu, le CSS pour le visuel et JavaScript pour les interactions.
Grâce à eux, les sites peuvent être dynamiques et réactifs sans avoir besoin de recharger la page en permanence et ils sont lus par des navigateur web.

Navigateurs


Un navigateur web, c’est un logiciel qui permet d’accéder à Internet et d’afficher les sites web.
Il traduit le code des pages (HTML, CSS, JavaScript) en un contenu visuel que l’on peut lire et utiliser.
Les plus connus sont Google Chrome, Firefox, Safari et Edge.

Langages serveur

Les langages serveur sont des langages qui s’exécutent sur un serveur pour gérer les données, les utilisateurs et générer des pages web avant de les envoyer au navigateur.
Contrairement aux langages client(comme HTML,CSS,JavaScript), qui s’exécutent directement sur l’ordinateur ou le téléphone de l’utilisateur, les langages serveur traitent les informations en arrière-plan.
Des exemples connus sont PHP, Python, Java et Node.js, qui permettent de créer des sites dynamiques et interactifs.

Frameworks

Un framework est un ensemble d’outils et de règles qui facilite le développement d’applications en offrant une structure prête à l’emploi.
Il permet de gagner du temps, d’écrire un code plus organisé et de respecter les bonnes pratiques.
Par exemple, Symfony et Zend sont utilisés pour PHP, Java EE pour Java et Django pour Python

Bases de données

Les bases de données sont des systèmes qui stockent, organisent et gèrent des informations de manière structurée, souvent utilisées avec des langages serveur pour récupérer ou modifier les données.
Elles permettent un accès rapide aux informations et facilitent leur gestion, pour communiquer avec ces logiciels, on utilise un langage : SQL.
Par exemple, MySQL, PostgreSQL ou SQL server sont des bases de données sont des systèmes de gestion de bases de données populaires .

Site responsive

Un site responsive est un site web conçu pour s’adapter automatiquement à la taille de l’écran sur lequel il est affiché, que ce soit un ordinateur, une tablette ou un smartphone.
Cela garantit une bonne expérience utilisateur, peu importe l’appareil utilisé.

Application mobile

Une application mobile est un logiciel conçu spécifiquement pour être utilisé sur des appareils mobiles comme les smartphones ou les tablettes.
Elle peut être téléchargée depuis des stores comme l'App Store ou Google Play et offre des fonctionnalités variées, allant de la gestion de tâches à des jeux ou des services de messagerie.
Contrairement aux sites web, ils sont optimisées pour fonctionner de manière fluide sur les petits écrans.

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Les serveurs sont des ordinateurs spéciaux qui stockent les sites web et les rendent accessibles à tous via Internet.
Quand un développeur crée un site, il utilise souvent un serveur local sur son propre ordinateur pour tester le site, ce qui permet de le voir avant qu'il ne soit en ligne.
Une fois le site prêt, il est envoyé sur un serveur réel, ce qui le rend accessible à tout le monde, grâce à des câbles de fibre optique très rapides qui transmettent les informations.

Adresse IP

Une adresse IP (Internet Protocol) est un identifiant unique attribué à chaque appareil connecté à Internet, comme un ordinateur, un téléphone ou un serveur, il s’agit d’une suite de nombres comme 255.69.577.66.
Elle permet aux appareils de se repérer et de communiquer entre eux sur le réseau.
Comme une adresse postale, l'adresse IP permet de savoir où envoyer les informations sur Internet, que ce soit pour accéder à un site web ou envoyer des données d’un appareil à un autre.

Nom d’hôte et DNS

Un nom d’hôte est un nom facile à retenir, comme "google.com", qui représente un site ou un service sur Internet.
Mais les ordinateurs communiquent entre eux via des adresses IP, bien plus complexes à retenir.
C’est là qu’intervient le DNS (Domain Name System) : un service d’annuaire qui traduit ces noms d’hôtes en adresses IP, permettant ainsi aux appareils de se retrouver et de se connecter correctement sans que l'on ait à se soucier de ces détails techniques.

Protocoles

Les protocoles sont des langages qui permettent aux machines de communiquer sur Internet.
Le TCP est un protocole bas niveau essentiel pour toutes les communications, comme les pages web ou les e-mails.
Des protocoles haut niveau, comme HTTP pour les pages web ou SMTP pour les e-mails, reposent souvent sur TCP pour échanger des données de manière structurée.

Partie 4 - Identifiez les métiers du web

Développeurs

Un développeur est un professionnel du numérique qui conçoit, écrit et teste du code informatique pour créer des logiciels, des sites web ou des applications, en utilisant des outils comme Visual Studio Code.
Il maîtrise différents langages de programmation (HTML,CSS et JavaScript), travaille souvent en équipe avec des outils comme Git pour la gestion de versions, afin de répondre aux besoins des utilisateurs ou des entreprises.

Product Managers et chefs de projets

Product Manager : Responsable de la vision et du développement d’un produit, il veille à ce qu’il réponde aux besoins des utilisateurs et aux objectifs de l’entreprise, en collaborant avec les équipes techniques et marketing.
Chef de projet : Il planifie, organise et coordonne un projet pour assurer son bon déroulement dans les délais et budgets impartis, en facilitant la communication entre les équipes.

Designers

Le designer conçoit l’apparence et l’expérience utilisateur d’un produit (site web, application, logo, etc.) en alliant esthétique et fonctionnalité.
Il utilise des outils comme Figma, Adobe XD ou Photoshop pour créer des interfaces attrayantes et intuitives.

Testeurs

Le testeur vérifie la qualité et la fiabilité d’un logiciel en identifiant les bugs et en s’assurant qu’il fonctionne correctement.
Il utilise des outils comme Selenium, JIRA ou Postman pour automatiser les tests et suivre les anomalies.

Techniciens informatique

Le technicien informatique installe, maintient et dépanne les équipements informatiques et réseaux pour garantir leur bon fonctionnement dans les entreprises.
Il utilise des outils comme Windows Server, Cisco Packet Tracer ou des logiciels de diagnostic pour résoudre les problèmes techniques.

Langages de développement web

HTML

Front-End

Squelette du site Web. Il définit la structure de base du contenu, comme les titres, les paragraphes, les images, etc.

CSS

Front-End

CSS est utilisé en front-end pour styliser les pages web, en définissant des éléments comme les couleurs, les polices et la mise en page et rend un site visuellement attrayant.

JavaScript

Front-End

JavaScript est un langage front-end qui permet d'ajouter des interactions et des animations sur les pages web, rendant ainsi le site plus dynamique sans recharger la page.

JavaScript

Back-End

En backend, JavaScript, via Node.js, permet de gérer des requêtes, interagir avec des bases de données et traiter des informations côté serveur, tout en utilisant le même langage que pour le front-end.

PHP

Back-End

PHP est un langage serveur utilisé pour créer des pages web dynamiques, interagir avec des bases de données et gérer des requêtes côté serveur. Il est utilisé pour le développement web et d’applications.

Java

Back-End

Java est un langage robuste utilisé pour développer des applications serveur, gérer des requêtes et interagir avec des bases de données. Il est apprécié pour sa stabilité et sa scalabilité dans les grandes applications

Python

Back-End

Python est un langage utilisé pour développer des applications serveur, gérer des requêtes et interagir avec des bases de données. Il est apprécié pour sa simplicité et sa rapidité de développement.

C#

Back-End

C# est un langage utilisé pour développer des applications serveur, souvent avec le framework ASP.NET, pour gérer des requêtes et interagir avec des bases de données. Il est populaire pour sa performance et son intégration avec les systèmes Windows.

SQL

Bases de données

SQL est un langage utilisé pour gérer et interagir avec des bases de données relationnelles. Il permet de créer, modifier, récupérer et supprimer des données à l'aide de requêtes simples ou complexes.

Boîte à outils du développeur

  • IDE comme VS Code : Un IDE comme VS Code est un logiciel qui aide les développeurs à écrire et organiser leur code.
    Il inclut des outils pour faciliter l’édition, le débogage et la gestion des projets.
  • Github : GitHub est une plateforme qui permet aux développeurs de stocker et de partager leur code en ligne.
    Elle facilite la collaboration sur des projets en permettant de suivre les changements et de gérer les versions du code.
  • Librairies / bibliothèques : Les librairies, ou bibliothèques, sont des collections de code pré-écrit que les développeurs peuvent utiliser pour ajouter des fonctionnalités à leurs projets sans avoir à tout coder depuis zéro.
    Elles facilitent le développement en offrant des outils et des solutions prêtes à l'emploi.
  • Frameworks : Un framework est un ensemble d’outils qui facilite le développement en fournissant une structure prête à l’emploi. Il permet de coder plus rapidement et proprement.
    Par exemple, Symfony pour PHP, Django pour Python et Java EE pour Java.
